Remote PHASH

Downloads each URL to /downloads (stable cache basename), runs hasher PHASH, and stores OSHASH / duration in cache/scenes.json. OSHASH is taken from HTTP Range when possible. URLs run one after another. Set REMOTE_HASH_DELETE_FILE=0 to keep the file after hashing. If you already have a PHASH (and optional OSHASH), use the form below to attach them to a scene without downloading.

Known PHASH → StashDB (no download)

Paste a 16-character perceptual hash, the StashDB scene URL, and duration in seconds — same submitFingerprint path as after a remote run. Optional OSHASH is included when provided (16 hex).
