MD5 → StashDB

Paste one MD5 per line (32 hex chars; colons/spaces are stripped). Each row is looked up on StashDB. Batch submit only sends rows that already matched a scene (same as Remote OSHASH). Unmatched rows need a pasted scene URL + duration per row. Submits use STASHDB_AUTO_CONTRIBUTE + API key.
